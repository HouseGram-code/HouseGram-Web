# Исправление MEGA Storage

## Проблема
Ваш текущий MEGA аккаунт возвращает ошибку 402 (Payment Required), что означает:
- Аккаунт заблокирован или требует оплаты
- Неверные учетные данные
- Превышен лимит бесплатного аккаунта

## Решение: Создать новый бесплатный MEGA аккаунт

### Шаг 1: Создайте новый аккаунт MEGA
1. Перейдите на https://mega.nz/register
2. Зарегистрируйте новый аккаунт (бесплатно 20GB!)
3. Подтвердите email

### Шаг 2: Обновите .env.local
Замените в файле `.env.local`:

```env
NEXT_PUBLIC_MEGA_EMAIL=ваш_новый_email@gmail.com
NEXT_PUBLIC_MEGA_PASSWORD=ваш_новый_пароль
```

### Шаг 3: Перезапустите приложение
```bash
npm run dev
```

## Преимущества MEGA:
- ✅ 20GB бесплатно (vs Firebase 5GB)
- ✅ Поддержка файлов до 4GB
- ✅ Не требует кредитной карты
- ✅ Автоматическое шифрование
- ✅ Уже интегрировано в ваш проект

## Альтернатива: Cloudflare R2 (Бесплатно 10GB)

Если MEGA не работает, можно использовать Cloudflare R2:
- 10GB хранилища бесплатно
- Без платы за трафик
- S3-совместимый API
