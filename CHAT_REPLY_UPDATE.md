# ✨ Обновление: Красивая кнопка "Ответить" в чате

## 🎯 Что добавлено:

### 1. **Кнопка "Ответить" в контекстном меню**
- Добавлена кнопка "Ответить" при долгом нажатии на сообщение
- Красивая иконка Reply из lucide-react
- Плавная анимация при открытии меню

### 2. **Блок предпросмотра ответа (Reply Preview)**
Создан в стиле Telegram с:
- ✅ Плавной анимацией появления/исчезновения
- ✅ Градиентным фоном (from-gray-50 to-white)
- ✅ Цветной левой границей (использует themeColor)
- ✅ Иконкой Reply
- ✅ Именем отправителя (жирным шрифтом, цвет темы)
- ✅ Превью текста сообщения (обрезается до 100 символов)
- ✅ Кнопкой закрытия (X) с hover эффектом
- ✅ Тенью для объёма

### 3. **Улучшенный UX**
- Автоматический фокус на поле ввода при нажатии "Ответить"
- Изменение placeholder на "Ответить..." когда активен режим ответа
- Автоматическая очистка блока ответа после отправки сообщения
- Плавные переходы и анимации

### 4. **Визуальный дизайн**
```
┌─────────────────────────────────────┐
│ [Reply Icon] Имя отправителя        │ ← Цвет темы
│              Текст сообщения...  [X]│
└─────────────────────────────────────┘
  ↑ Цветная граница (themeColor)
```

## 📝 Технические детали:

### Добавленные состояния:
```typescript
const [replyingTo, setReplyingTo] = useState<{
  messageId: string;
  senderName: string;
  text: string;
  senderAvatar?: string;
} | null>(null);
```

### Новые функции:
- `handleReply()` - обработка нажатия на кнопку "Ответить"
- `cancelReply()` - отмена ответа

### Анимации:
- Использован `framer-motion` для плавного появления/исчезновения
- `AnimatePresence` для управления монтированием/размонтированием
- Transition duration: 0.2s для быстрого отклика

## 🎨 Стилизация:

### Цвета:
- Фон: `bg-gradient-to-r from-gray-50 to-white`
- Граница: `border-l-4` с цветом темы
- Текст имени: цвет темы (themeColor)
- Текст сообщения: `text-gray-600`
- Иконки: `text-gray-400`

### Эффекты:
- Тень: `shadow-sm`
- Скругление: `rounded-lg`
- Hover на кнопке закрытия: `hover:bg-gray-100`
- Transition: `transition-all`

## 🔄 Интеграция:

Блок ответа интегрирован:
1. Над полем ввода
2. Внутри контейнера с backdrop-blur
3. С border-top для визуального разделения
4. С отступами для красивого отображения

## 📱 Адаптивность:

- Работает на всех размерах экранов
- Текст обрезается с `truncate` если слишком длинный
- Кнопка закрытия всегда видна (shrink-0)
- Минимальная ширина не задана для гибкости

## ✅ Результат:

Теперь чат имеет полноценную функцию ответа на сообщения с красивым UI в стиле Telegram!
