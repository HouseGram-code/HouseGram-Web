# Matrix Integration для HouseGram

## 🚀 Что добавлено

Интеграция с Matrix Protocol позволяет вашему мессенджеру:

### ✅ Основные возможности:
- **Автоматическая настройка** - умный выбор лучшего сервера
- **Децентрализованное общение** - подключение к любому Matrix серверу
- **Большие файлы** - поддержка файлов до 50MB (вместо 2GB Telegram)
- **Медиафайлы** - отправка фото, видео, аудио, документов
- **Истории** - создание и просмотр Matrix комнат
- **Шифрование** - end-to-end шифрование по умолчанию
- **Кроссплатформенность** - общение с пользователями других Matrix клиентов
- **Автоматическое переподключение** - восстановление сессии при перезапуске

### 🔧 Технические детали:
- **Matrix JS SDK** - официальная библиотека для работы с Matrix
- **Автоматическое сжатие** - оптимизация изображений и видео
- **Реальное время** - мгновенная доставка сообщений
- **Офлайн поддержка** - синхронизация при восстановлении соединения
- **Умный выбор сервера** - автоматическое тестирование доступности

## 📱 Как использовать

### 1. Доступ к Matrix
1. Откройте боковое меню (☰)
2. Выберите **"Matrix Rooms"**
3. Система автоматически найдет лучший сервер

### 2. Автоматическая настройка
1. **Умный выбор сервера:**
   - Система тестирует популярные серверы
   - Выбирает самый быстрый и доступный
   - Показывает процесс в реальном времени

2. **Авторизация:**
   - Войдите в существующий аккаунт
   - Или создайте новый аккаунт
   - Данные сохраняются для автоматического входа

### 3. Работа с комнатами
- **Создать комнату** - публичную или приватную
- **Присоединиться** - по ID или алиасу комнаты
- **Поиск комнат** - быстрый поиск по названию

### 4. Отправка медиафайлов
В Matrix чате доступны:
- 📷 **Изображения** - автоматическое сжатие
- 🎥 **Видео** - поддержка всех форматов
- 🎵 **Аудио** - музыка и голосовые сообщения
- 📄 **Файлы** - документы до 50MB

## 🔒 Безопасность

### Приватность:
- **Децентрализация** - нет единого сервера-владельца
- **Шифрование** - все сообщения зашифрованы
- **Открытый код** - можно проверить безопасность
- **Контроль данных** - выбор сервера для хранения

### Рекомендации:
- Используйте надежные серверы
- Включите двухфакторную аутентификацию
- Регулярно обновляйте пароли
- Проверяйте ключи шифрования

## 🌐 Популярные Matrix серверы

### Официальные:
- **matrix.org** - основной сервер Matrix Foundation
- **matrix-client.matrix.org** - сервер Element

### Европейские:
- **tchncs.de** - немецкий сервер с фокусом на приватность

### Специализированные:
- **mozilla.org** - сервер Mozilla
- **kde.org** - сервер KDE проекта

## 🛠️ Технические требования

### Минимальные:
- Современный браузер с WebRTC
- Стабильное интернет-соединение
- 50MB свободного места для кэша

### Рекомендуемые:
- Chrome/Firefox последней версии
- Скорость интернета от 1 Мбит/с
- 100MB свободного места

## 🐛 Устранение неполадок

### Частые проблемы:

**1. Не удается подключиться к серверу**
- Проверьте URL сервера
- Убедитесь в стабильности интернета
- Попробуйте другой сервер

**2. Ошибка при загрузке файлов**
- Проверьте размер файла (макс. 50MB)
- Убедитесь в поддержке формата
- Проверьте настройки сервера

**3. Сообщения не доставляются**
- Проверьте статус подключения
- Обновите страницу
- Перезайдите в аккаунт

### Логи и отладка:
- Откройте Developer Tools (F12)
- Проверьте вкладку Console
- Найдите ошибки Matrix клиента
