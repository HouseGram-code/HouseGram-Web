# 🚀 Быстрая настройка MEGA Storage

## ✅ Что получили

Ваш мессенджер теперь поддерживает:
- 📦 **20GB бесплатного хранилища**
- 📁 **Файлы до 4GB** (фото, видео, музыка)
- ⚡ **Быстрая загрузка** через CDN
- 🔒 **Шифрование** файлов
- ♾️ **Без ограничений** навсегда

---

## 🎯 Настройка за 3 шага

### Шаг 1: Создайте MEGA аккаунт

1. Откройте [https://mega.nz](https://mega.nz)
2. Нажмите **"Создать аккаунт"**
3. Заполните:
   - Email: `ваш-email@example.com`
   - Пароль: `минимум 8 символов`
   - Имя: `ваше имя`
4. Подтвердите email
5. Готово! У вас есть 20GB бесплатно! 🎉

### Шаг 2: Добавьте данные в проект

Откройте файл `.env.local` и замените:

```env
# MEGA Storage Configuration
NEXT_PUBLIC_MEGA_EMAIL=ваш-email@example.com
NEXT_PUBLIC_MEGA_PASSWORD=ваш-пароль
```

⚠️ **Важно**: Используйте РЕАЛЬНЫЕ данные от вашего MEGA аккаунта!

### Шаг 3: Запустите приложение

```bash
npm run dev
```

Откройте браузер и увидите:
```
🚀 Initializing MEGA Storage...
✅ MEGA Storage initialized successfully
📦 Account: Ваше Имя
💾 Storage used: 0 GB / 20 GB
```

---

## 🎮 Как использовать

1. Откройте Matrix чат
2. Нажмите на скрепку 📎
3. Выберите файл (до 4GB!)
4. Файл автоматически загрузится в MEGA
5. Получатель увидит файл и сможет скачать

| Тип файла | Форматы | Макс. размер |
|-----------|---------|--------------|

## ❓ Проблемы?

### "MEGA credentials not configured"
➡️ Добавьте email и пароль в `.env.local`

### "Login failed"
➡️ Проверьте правильность email и пароля
➡️ Подтвердите email в MEGA

### "File too large"
➡️ Максимальный размер файла: 4GB
