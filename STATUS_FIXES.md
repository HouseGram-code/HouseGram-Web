# Исправления статусов и аватаров

## Что было исправлено

### 1. Статус пользователей

**Проблема:** Все пользователи показывались как "в сети", даже если они не заходили на сайт.

**Решение:**
- Добавлена проверка реального статуса из Firebase (`status` и `lastSeen`)
- Статус "в сети" показывается только если `status === 'online'`
- Для офлайн пользователей показывается время последнего визита:
  - "был(а) только что" - меньше 1 минуты
  - "был(а) X мин. назад" - меньше часа
  - "был(а) X ч. назад" - меньше суток
  - "был(а) X дн. назад" - больше суток
  - "был(а) недавно" - если нет данных

**Где изменено:**
- `context/ChatContext.tsx` - логика определения статуса при загрузке контактов
- `components/ChatView.tsx` - отображение `statusOffline` вместо `statusOnline`

### 2. Статус бота

**Проблема:** Бот показывался как "в сети", хотя должен быть "бот".

**Решение:**
- Изменен `statusOnline` с "в сети" на "бот"
- Теперь бот всегда показывает статус "бот", как в Telegram

**Где изменено:**
- `lib/mockData.ts` - изменен статус для `test_bot`

### 3. Цвета аватаров

**Проблема:** Все пользователи без аватарки имели одинаковый серый цвет.

**Решение:**
- Добавлена функция `getAvatarColor(userId)` для генерации уникального цвета
- 15 ярких цветов в палитре
- Цвет генерируется на основе хеша ID пользователя (всегда одинаковый для одного пользователя)

**Палитра цветов:**

**Где изменено:**
- `components/ChatList.tsx` - добавлена функция и использование в поиске
- `context/ChatContext.tsx` - использование при загрузке контактов

## Как это работает

### Обновление статуса

При входе пользователя:
```typescript
await updateDoc(doc(db, 'users', userId), { 
  status: 'online', 
  lastSeen: serverTimestamp() 
});
```

При выходе/закрытии вкладки:
```typescript
await updateDoc(doc(db, 'users', userId), { 
  status: 'offline', 
  lastSeen: serverTimestamp() 
});
```

### Отображение статуса

### Генерация цвета аватара

```typescript
const getAvatarColor = (userId: string) => {
  let hash = 0;
  for (let i = 0; i < userId.length; i++) {
    hash = userId.charCodeAt(i) + ((hash << 5) - hash);
  }
  return colors[Math.abs(hash) % colors.length];
};
```

## Результат

✅ Пользователи показывают реальный статус (в сети / был(а) X назад)
✅ Бот показывает статус "бот"
✅ Каждый пользователь без аватарки имеет уникальный яркий цвет
✅ Цвет аватара стабильный (не меняется при перезагрузке)
✅ Красивые, различимые цвета как в Telegram

## Дополнительно

Система автоматически обновляет статус при:
- Входе в приложение
- Переключении вкладок (visibility change)
- Закрытии браузера (beforeunload)

Все изменения сохраняются в Firebase и синхронизируются между устройствами.
