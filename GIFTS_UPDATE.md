# Обновление системы подарков

## Что нового

### 1. Кнопка "Отправить себе подарок"
- Добавлена в раздел "Молнии" (StarsView)
- Позволяет отправить подарок самому себе для тестирования
- Открывает раздел "Мои подарки"

### 2. Расширенный каталог подарков
Теперь доступно 8 видов подарков:

| Подарок | Эмодзи | Стоимость |
|---------|--------|-----------|
| Плюшевый мишка | 🧸 | 15 ⚡ |
| Красное сердце | ❤️ | 10 ⚡ |
| Роза | 🌹 | 12 ⚡ |
| Торт | 🎂 | 18 ⚡ |
| Звезда | ⭐ | 20 ⚡ |
| Подарочная коробка | 🎁 | 25 ⚡ |
| Бриллиант | 💎 | 50 ⚡ |
| Корона | 👑 | 100 ⚡ |

### 3. Улучшенный интерфейс выбора подарков
- Подарки отображаются в виде сетки 2x4 (как в Telegram)
- Показывается текущий баланс пользователя
- Недоступные подарки (недостаточно молний) отображаются с пометкой
- Анимация при наведении (масштабирование)

### 4. Исправлено отображение подарков в чате
**Проблема:** Подарки отправлялись только в Supabase, но не отображались в чате

**Решение:**
- Подарки теперь сохраняются в обе БД (Firebase и Supabase)
- Обновляется последнее сообщение в чате
- Обновляется баланс отправителя и получателя в обеих БД
- Обновляется статистика подарков

### 5. Отображение подарков в чате
Подарки отображаются как специальные сообщения:
- Градиентный фон (фиолетовый → розовый)
- Большой эмодзи с анимацией (покачивание, вращение)
- Название подарка
- Информация об отправителе

## Как работает

### Отправка подарка

1. Пользователь открывает раздел "Молнии"
2. Нажимает "Отправить подарок"
3. Выбирает получателя из списка контактов
4. Выбирает подарок из каталога
5. Подтверждает отправку

### Что происходит при отправке

1. **Проверка баланса** - достаточно ли молний
2. **Создание чата** (если не существует) в обеих БД
3. **Отправка сообщения** с подарком в Firebase
4. **Сохранение подарка** в Supabase (таблица `received_gifts`)
5. **Обновление баланса** отправителя (-стоимость подарка)
6. **Обновление статистики**:
   - `giftsSent` у отправителя +1
   - `giftsReceived` у получателя +1
7. **Показ уведомления** об успешной отправке

## Технические детали

### Firebase структура сообщения с подарком
```javascript
{
  chatId: string,
  senderId: string,
  text: '',
  createdAt: timestamp,
  status: 'sent',
  type: 'sent',
  gift: {
    id: string,
    name: string,
    emoji: string,
    cost: number,
    from: string
  }
}
```

### Supabase таблица `received_gifts`
```sql
{
  user_id: uuid,
  gift_id: string,
  name: string,
  emoji: string,
  cost: integer,
  from_user_id: uuid,
  from_name: string,
  can_convert: boolean,
  received_at: timestamp
}
```
