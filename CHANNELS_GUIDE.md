# Руководство по использованию каналов

## Как создать канал

1. Откройте боковое меню (кнопка ☰ в левом верхнем углу)
2. Нажмите "Создать канал" (иконка радио 📻)
3. Заполните форму:
   - **Название** (обязательно) - до 50 символов
   - **Описание** (необязательно) - до 200 символов
   - **URL аватарки** (необязательно) - ссылка на изображение
4. Нажмите "Создать канал"
5. Канал появится в списке чатов

## Как отправить пост в канал

1. Откройте свой канал из списка чатов
2. Введите текст в поле ввода внизу
3. Нажмите кнопку отправки (✈️)
4. Пост появится в канале

**Важно:** Отправлять посты может только владелец канала (тот кто его создал)

## Просмотр информации о канале

1. Откройте канал
2. Нажмите кнопку Info (ℹ️) в правом верхнем углу
3. Вы увидите:
   - Название и аватарку канала
   - Количество подписчиков
   - Описание канала
   - Ссылку на канал (нажмите чтобы скопировать)

## Счетчик просмотров

- Каждый пост в канале имеет счетчик просмотров
- Иконка глаза (👁️) показывает сколько раз пост был просмотрен
- Просмотр засчитывается автоматически при открытии канала
- Один пользователь = один просмотр (повторные не считаются)

## Особенности каналов

- ✅ Каналы отображаются в общем списке чатов
- ✅ Показывается количество подписчиков вместо статуса
- ✅ Последний пост отображается в превью
- ✅ Автоматический счетчик просмотров для каждого поста
- ✅ Уникальная ссылка для каждого канала

## Технические требования

- Авторизация в системе
- Активный аккаунт (не забанен)
- Обновленные правила Firestore

## Устранение неполадок

### Канал не появляется в списке
- Обновите страницу (F5)
- Проверьте что вы авторизованы
- Откройте консоль браузера (F12) и проверьте ошибки

### Не могу отправить пост
- Убедитесь что вы владелец канала
- Проверьте что правила Firestore обновлены
- Проверьте консоль браузера на ошибки

### Не работает счетчик просмотров
- Обновите правила Firestore
- Проверьте что вы не владелец канала (свои просмотры не считаются)
- Откройте канал с другого аккаунта для теста
