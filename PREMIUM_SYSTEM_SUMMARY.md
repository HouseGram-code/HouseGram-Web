## 🛠 Технические детали

### Новые компоненты:
1. **PremiumView.tsx** - Страница покупки Premium
2. **PremiumBadge.tsx** - Анимированный значок Premium
3. **PremiumModal.tsx** - Модальное окно с информацией о Premium

### Обновленные компоненты:
1. **ChatContext.tsx** - Добавлена логика Premium и лимитов ИИ
2. **AdminView.tsx** - Добавлена вкладка управления Premium
3. **SettingsView.tsx** - Добавлено отображение статуса Premium и лимитов ИИ
4. **ChatList.tsx** - Добавлены Premium значки в списке контактов
5. **ChatView.tsx** - Добавлен Premium значок в заголовке чата

### База данных:
- **premium_requests** - Коллекция заявок на Premium
- **users.premium** - Флаг Premium статуса
- **users.premiumExpiry** - Дата истечения Premium
- **users.aiRequestsToday** - Счетчик ИИ запросов по дням

## 🔄 Процесс покупки Premium

1. Пользователь заходит в Настройки → HouseGram Premium
2. Нажимает "Купить Premium за 299₽"
3. Создается заявка в коллекции `premium_requests`
4. Пользователю показывается инструкция связаться с @HouseGramBot
5. Администратор видит заявку в админ панели
6. Администратор одобряет заявку → Premium активируется на 30 дней
7. Пользователь получает Premium значок и увеличенные лимиты ИИ
