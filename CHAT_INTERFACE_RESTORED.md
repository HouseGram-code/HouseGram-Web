# ✅ Восстановлен старый интерфейс чата

## Что было изменено:

### 1. **Упрощено поле ввода** 
- Убрана сложная структура с вложенными div
- Возвращен простой однострочный input
- Убрана поддержка многострочного ввода (textarea)
- Кнопки расположены в одну линию с полем ввода

### 2. **Удалена функция "Ответить"**
- Убрано состояние `replyingTo`
- Удалена функция `handleReply()`
- Удалена функция `cancelReply()`
- Убрана кнопка "Ответить" из контекстного меню
- Убран блок отображения цитируемого сообщения над полем ввода

### 3. **Упрощена логика отправки**
- Убраны параметры ответа из `sendMessage()`
- Убрана логика статуса "печатает" (typing status)
- Упрощен код обработки отправки сообщения

### 4. **Визуальные изменения**
- Поле ввода теперь выровнено по центру (`items-center` вместо `items-end`)
- Убраны отступы `mb-0.5` у кнопок
- Кнопка отправки и микрофон теперь одинакового размера (9x9 вместо 10x10)
- Упрощены стили для более чистого вида

## Результат:

Теперь интерфейс чата выглядит как классический Telegram:
- Простое поле ввода в одну строку
- Кнопки прикрепления, эмодзи и отправки/микрофона в одну линию
- Нет функции ответа на сообщения
- Чистый и минималистичный дизайн

## Файлы изменены:
- `components/ChatView.tsx` - основной файл чата

## Примечание:
Компоненты `ChatInput.tsx` и `MessagesContainer.tsx` остались без изменений, но не используются в текущей версии.
