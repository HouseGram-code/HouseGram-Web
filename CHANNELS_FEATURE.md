# Функционал каналов

## Что добавлено

### 1. Создание канала
- Кнопка "Создать канал" в боковом меню (иконка Radio)
- Форма создания с полями:
  - Название канала (обязательно, до 50 символов)
  - Описание (необязательно, до 200 символов)
  - URL аватарки (необязательно)
- Автоматическая генерация ссылки на канал

### 2. Просмотр канала
- Каналы отображаются в списке чатов
- Показывается количество подписчиков вместо статуса
- При открытии канала:
  - Посты отображаются как сообщения
  - Счетчик просмотров (иконка глаза) для каждого поста
  - Кнопка информации (Info) в заголовке вместо меню

### 3. Информация о канале
- Название и аватарка
- Количество подписчиков
- Описание канала
- Ссылка на канал (копируется по клику)
- Заглушка "Настройки скоро..."

### 4. Отправка постов
- Владелец канала может отправлять посты через обычный интерфейс чата
- Посты автоматически получают счетчик просмотров
- Просмотры увеличиваются автоматически при открытии канала

## Технические детали

### Структура данных

#### Канал (channels/{channelId})
```typescript
{
  id: string;
  name: string;
  description?: string;
  avatarUrl?: string;
  createdBy: string; // UID создателя
  createdAt: Timestamp;
  subscribersCount: number;
  link: string;
  subscribers: string[]; // Массив UID подписчиков
}
```

#### Пост канала (channels/{channelId}/posts/{postId})
```typescript
{
  text: string;
  time: string;
  createdAt: Timestamp;
  senderId: string;
  views: number;
  viewedBy: string[]; // Массив UID просмотревших
  // + опциональные поля для медиа
}
```

### Новые компоненты
- `CreateChannelView.tsx` - форма создания канала
- `ChannelInfoView.tsx` - информация о канале

### Изменения в существующих файлах
- `types/index.ts` - добавлен тип Channel, поле views в Message
- `ChatView.tsx` - отображение просмотров, кнопка Info для каналов
- `ChatContext.tsx` - логика загрузки каналов и постов, отправка постов
- `SideMenu.tsx` - кнопка создания канала
- `app/page.tsx` - роутинг новых view
- `firestore.rules` - правила безопасности для каналов

### Правила безопасности
- Любой авторизованный пользователь может создать канал
- Только владелец канала может отправлять посты
- Любой может подписаться/отписаться
- Просмотры увеличиваются автоматически (один раз на пользователя)
- Только владелец или админ может удалить канал/пост
