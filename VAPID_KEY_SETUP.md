# Как получить правильный VAPID ключ

## Проблема
Текущий VAPID ключ невалидный. Нужно сгенерировать новый в Firebase Console.

## Решение

### Шаг 1: Откройте Firebase Console
1. Перейдите на https://console.firebase.google.com
2. Выберите проект **housegram-d070d**

### Шаг 2: Перейдите в настройки Cloud Messaging
1. Нажмите на шестеренку ⚙️ (Settings) в левом верхнем углу
2. Выберите **Project settings**
3. Перейдите на вкладку **Cloud Messaging**

### Шаг 3: Сгенерируйте Web Push certificates
1. Прокрутите вниз до раздела **Web Push certificates**
2. Если ключ уже есть - скопируйте его
3. Если ключа нет - нажмите **Generate key pair**
4. Скопируйте сгенерированный ключ

### Шаг 4: Обновите .env.local
1. Откройте файл `.env.local`
2. Найдите строку:
   ```
   NEXT_PUBLIC_FIREBASE_VAPID_KEY=BJOZ6Oy5f1674RmQZYJuy5ctJtyNW7vNST6MOqnYskESh9NVlb19Ucez-V75y1xLfytaV9MFuceipOcheQE5ppcApr
   ```
3. Замените на новый ключ:
   ```
   NEXT_PUBLIC_FIREBASE_VAPID_KEY=ваш-новый-vapid-ключ
   ```

### Шаг 5: Перезапустите сервер
```bash
npm run dev
```

### Шаг 6: Раскомментируйте код
В файле `context/ChatContext.tsx` найдите закомментированный блок (строка ~147):
```typescript
// Инициализация push-уведомлений (временно отключено)
// TODO: Получить правильный VAPID ключ из Firebase Console
/*
if (typeof window !== 'undefined' && 'Notification' in window) {
  ...
}
*/
```

Раскомментируйте его (уберите `/*` и `*/`).

## Проверка

После всех шагов:
1. Обновите страницу (F5)
2. Откройте консоль (F12)
3. Должно появиться: "FCM Token: ..."
4. Если ошибка - проверьте что ключ скопирован полностью

## Альтернатива (если не работает)

Можно использовать приложение без push-уведомлений:
- Обычные уведомления в браузере работают
- Звуки работают
- Просто не будет уведомлений когда браузер закрыт

Все остальные функции (каналы, сообщения, статистика) работают нормально!
